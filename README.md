# mellow-client

## Project setup

```
npm install
```

## Setup Environment

Create a file called .env.development with following data

```
VUE_APP_FACILIO_BASE_URL=http://localhost:3000
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
